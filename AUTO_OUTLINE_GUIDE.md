# 🚀 自动生成大纲功能使用指南

## 功能概述

"自动生成大纲"功能允许你无需拆书，直接生成完整的小说大纲，支持超长篇体量（最多1000章，每章3000字）。

## ✨ 功能特点

### 核心优势
- ✅ **无需拆书**：不需要提供参考书籍，直接从零开始生成
- ✅ **超长篇支持**：最多支持1000章，每章可达3000字（总字数300万字）
- ✅ **完整内容**：自动生成世界观、人物、故事大纲、章节概要
- ✅ **实时进度**：流式输出，实时显示生成进度
- ✅ **智能分批**：章节概要分批生成，避免超时

### 生成内容
1. **世界观设定**
   - 核心设定
   - 力量体系
   - 地理环境
   - 社会结构
   - 历史背景
   - 核心冲突
   - 创新点

2. **主要人物**（3-5个）
   - 姓名、角色、年龄
   - 性格特点
   - 背景故事
   - 能力/特长
   - 目标/动机
   - 弱点/缺陷
   - 人物关系

3. **故事大纲**
   - 故事开端
   - 主要冲突
   - 转折点
   - 高潮
   - 结局

4. **章节概要**（可设置数量）
   - 每章标题
   - 每章概要（100-200字）

## 📖 使用步骤

### 步骤1: 打开自动生成大纲

1. 在左侧边栏找到"自动生成大纲"按钮
2. 点击按钮，弹出配置窗口

### 步骤2: 配置小说参数

#### 必填项
- **小说标题**：输入你的小说名称
- **小说类型**：选择类型
  - 玄幻
  - 奇幻
  - 科幻
  - 仙侠
  - 魔幻
  - 异能
  - 末世
- **目标章节数**：10-1000章
- **每章字数**：1000-10000字

#### 可选项
- **核心创意**：描述你的核心创意、独特设定或故事起点
- **其他要求**：其他特殊要求或偏好

### 步骤3: 开始生成

点击"开始生成"按钮，系统会：
1. 生成世界观设定（5%）
2. 生成主要人物（15%）
3. 生成故事大纲（20%）
4. 分批生成章节概要（20%-90%）
5. 完成（100%）

### 步骤4: 应用到大纲

生成完成后：
1. 查看"生成完成"提示
2. 检查生成的数据统计（章节数、角色数等）
3. 点击"应用到大纲"按钮

### 步骤5: 查看和应用

系统会自动填充以下字段：
- ✅ 小说标题
- ✅ 章节设置（章节数、每章字数）
- ✅ 世界观设定（世界观标签页）
- ✅ 主要人物（人物管理标签页）
- ✅ 故事大纲（故事大纲标签页）

## 🔧 技术细节

### API端点
```
POST /api/ai/generate-full-outline
```

### 请求参数
```json
{
  "title": "小说标题",
  "novelType": "玄幻",
  "targetChapterCount": 100,
  "targetWordCount": 3000,
  "coreIdea": "核心创意",
  "customRequirements": "其他要求"
}
```

### 响应格式（流式）
```
event: progress
data: {"phase": "世界观设定", "percentage": 5, "message": "正在生成世界观设定..."}

event: world
data: {"coreSetting": "...", "powerSystem": "...", ...}

event: characters
data: [{"name": "...", "role": "...", ...}]

event: story
data: {"beginning": "...", "mainConflict": "...", ...}

event: chapters
data: {"startChapter": 1, "endChapter": 50, "chapters": [...]}

event: complete
data: {"success": true, "totalChapters": 100, ...}
```

### 限制条件
- 章节数范围：10-1000章
- 每章字数：1000-10000字
- 单次请求超时：300秒（5分钟）
- Vercel免费计划限制

## ⚠️ 注意事项

### 1. 质量把控
- 生成的大纲仅供参考，需要人工审核
- 可以手动调整世界观、人物、大纲
- 章节概要需要根据实际创作情况调整

### 2. 章节规划
- 1000章的超长篇建议分阶段创作
- 每个阶段（如200章）可以重新生成后续大纲
- 灵活调整，不必严格遵循生成的大纲

### 3. 内容一致性
- 确保人物设定前后一致
- 保持世界观逻辑连贯
- 注意伏笔和呼应

### 4. 避免套路化
- 系统会尽量避免常见套路
- 但仍需人工检查，避免俗套情节
- 可以在"其他要求"中明确禁止某些套路

## 🆘 常见问题

### Q1: 生成的大纲质量如何？
A: 生成质量取决于核心创意的详细程度。建议详细描述你的想法，系统会根据你的描述生成更贴合的大纲。

### Q2: 可以修改生成的大纲吗？
A: 可以！生成的大纲会自动填充到对应字段，你可以手动修改任何部分。

### Q3: 1000章需要多长时间生成？
A: 大约40-60分钟，具体取决于AI响应速度。建议分阶段生成（如每200章一次）。

### Q4: 生成失败怎么办？
A: 系统会自动重试。如果持续失败，可以尝试减少章节数或简化核心创意。

### Q5: 可以同时生成多个大纲吗？
A: 可以多次打开弹窗，生成不同的大纲进行比较。
