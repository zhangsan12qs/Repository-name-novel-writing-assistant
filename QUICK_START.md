欢迎使用网络小说写作助手！本指南将帮助你在 5 分钟内开始使用。

## 📋 前置准备

### 方法一：本地运行（需要安装 Node.js）

#### 1. 检查环境

```bash
# 检查 Node.js 版本（需要 24.x 或更高）
node --version

# 如果未安装，请访问 https://nodejs.org 下载安装

# 检查 pnpm 版本
pnpm --version

# 如果未安装，运行：
npm install -g pnpm
```

#### 2. 安装依赖

```bash
cd projects
pnpm install
```

#### 3. 启动应用

```bash
# 开发模式（推荐）
pnpm dev

# 或生产模式
pnpm build
pnpm start
```

#### 4. 访问应用

浏览器打开：http://localhost:5000

### 方法二：在线使用（无需安装）

访问已部署的在线版本（需要管理员提供链接）

## 🎬 第一次使用

### 1. 创建你的小说

1. **输入小说标题**：在左上角输入框填写你的小说名称
2. **设置大纲**：点击"故事大纲"标签，填写你的小说大纲
3. **创建角色**：点击"人物管理"标签，添加你的主要角色

### 2. 开始写作

1. **创建章节**：在章节列表中，点击"+"添加新章节
2. **填写章节标题**：输入章节名称（如"第一章：觉醒"）
3. **开始写作**：在编辑区域输入你的内容

### 3. 使用 AI 辅助

#### 生成大纲

1. 在"故事大纲"标签下
2. 输入生成提示（格式：类型,主题,主角）
   - 示例：`玄幻,穿越,现代程序员`
3. 点击"生成大纲"按钮
4. 等待 AI 生成完整大纲

#### 生成角色

1. 在"人物管理"标签下
2. 输入角色描述（格式：角色定位,性格,背景）
   - 示例：`主角,冷静睿智,现代程序员穿越`
3. 点击"生成角色"按钮

#### 批量生成章节

1. 在"批量生成章节"区域
2. 设置章节数量（1-100章）
3. 设置每章字数（100-10000字）
4. 点击"⚡ 一键批量生成N章"按钮
5. 等待生成完成（可能需要较长时间）

### 4. 查看问题反馈

写作过程中，底部会自动显示问题提示：

- 🔴 **错误**：狗血剧情、感情线、成长线等（必须修改）
- 🟡 **警告**：华美空洞、流水账、逻辑问题（建议修改）
- 🔵 **提示**：写作建议（可参考）

### 5. 修复问题

#### 自动修复

1. 点击问题右侧的"一键更改"按钮
2. AI 会自动修复该问题
3. 查看修复结果，如不满意可重新修复

#### 手动修复

1. 根据提示信息手动修改内容
2. 修改后问题会自动更新

## 🚨 写作规矩

### 禁止的内容

1. ❌ **感情线作为主线**：严禁以恋爱为主题
2. ❌ **主角个人成长**：严禁以"变强、升级"为主线
3. ❌ **狗血剧情**：三角恋、退婚、打脸、种马等
4. ❌ **套路化**：重生、系统、无敌等老套情节

### 必须达到的标准

1. ✅ **评分 > 80分**：综合质量评分
2. ✅ **扣分 < 80分**：问题扣减分数
3. ✅ **字数达标**：每章至少达到目标字数
4. ✅ **内容紧凑**：避免平淡无趣，确保有悬念

### AI 写作弊端规避

1. ✅ **华美空洞**：避免堆砌形容词，内容要有实质
2. ✅ **流水账**：避免"然后、接着"等连接词滥用
3. ✅ **逻辑 bug**：确保人设一致，前后不矛盾
4. ✅ **不推剧情**：每段必须推进主线
5. ✅ **内容注水**：删除无效描写，避免凑字数
6. ✅ **人物扁平**：增加人物性格描写和动机
7. ✅ **对话平淡**：增强对话张力和潜台词

## 🔧 高级技巧

### 1. 高效模式

当章节数超过 100 或人物数超过 50 时：
- 点击"高效模式"开关
- 优化检测频率，提升响应速度
- 保持核心质量检查功能

### 2. 批量生成策略

- 每次生成 20-50 章最佳
- 避免一次性生成太多（可能超时）
- 生成后手动检查剧情（建议每次不超过 50 章）

### 3. 数据管理

- 数据自动保存到 IndexedDB
- 刷新页面不会丢失数据
- 使用"数据管理中心"可查看和管理快照

### 4. 性能优化

- 启用高效模式
- 减少列表显示数量
- 定期清理旧数据

## 📞 获取帮助

### 常见问题

**Q: 数据丢失怎么办？**
A: 应用使用 IndexedDB 自动保存，刷新页面不会丢失。如仍丢失，使用"数据管理中心"恢复。

**Q: 生成速度慢？**
A: 尝试启用"高效模式"，或减少单次生成的章节数。

**Q: 质量检测不通过？**
A: AI 会自动重写直到达标。如多次重写仍失败，请检查大纲设定是否符合写作规矩。

**Q: 如何导出内容？**
A: 使用"数据管理中心"导出功能，或手动复制内容。
